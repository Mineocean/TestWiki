## 项目结构

```
src/content/docs/         ← 文档内容（丢 .mdx 即可）
├── _meta.ts              ← 侧边栏配置
├── getting-started.mdx
└── dev/
    ├── _meta.ts
    ├── architecture.mdx
    └── api-reference.mdx
```

---

## 本地开发

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

Astro 会在 `http://localhost:4321` 启动热重载开发服务器。

### 构建生产版本

```bash
npm run build
```

构建产物输出到 `dist/` 目录，为纯静态文件。

---

## 添加新文档

1. 在 `src/content/docs/` 目录创建 `.mdx` 文件
2. 在对应目录的 `_meta.ts` 中添加一行标题和路径
3. 完成，无需其他操作

```
# 1. 创建文件: src/content/docs/tutorial.mdx
---
title: 教程
---

## 第一步
...

# 2. 在 src/content/docs/_meta.ts 加一行:
{ label: '教程', slug: 'tutorial' }
```

## 部署

- **Vercel**: 自动检测 Astro
- **Netlify**: Build `npm run build`, Publish `dist`
- **Cloudflare Pages**: Build `npm run build`, Output `dist`
